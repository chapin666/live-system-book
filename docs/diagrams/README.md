# 图表绘制规范

本书使用**轻量级图表方案**，确保文档加载迅速。

---

## 推荐方案

| 场景 | 推荐方案 | 示例 |
|:---|:---|:---|
| 简单流程 | 文本箭头 | `A → B → C` |
| 架构图 | ASCII 文本框 | 见下方 |
| 数据对比 | Markdown 表格 | 见下方 |
| 复杂架构 | 外链 SVG | `<img src="..."/>` |

---

## 配色规范

如需彩色，统一使用：

| 颜色 | 用途 | 十六进制 |
|:---|:---|:---|
| 浅蓝 | 输入/输出 | `#e3f2fd` |
| 浅橙 | Demuxer | `#fff3e0` |
| 浅绿 | Decoder | `#e8f5e9` |
| 浅粉 | Renderer | `#fce4ec` |

## Markdown 表格示例

| 步骤 | 阶段 | 输入 | 输出 |
|:---|:---|:---|:---|
| 1 | 解封装 | MP4 文件 | H.264 数据包 |
| 2 | 解码 | H.264 数据 | YUV 像素帧 |
| 3 | 渲染 | YUV 像素 | 屏幕显示 |

## 文件存放

复杂图表存为 SVG：

```
docs/
├── images/          # SVG/PNG 图片
│   ├── arch.svg
│   └── ...
└── diagrams/        # Draw.io 源文件
    └── arch.drawio
```

Markdown 引用：
```markdown
<img src="docs/images/arch.svg" width="600"/>
```
