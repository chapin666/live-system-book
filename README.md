# 《直播连麦系统：从零到一》

> 一本手把手教你从零构建直播系统的实战教程

采用 **Pipeline 架构**，通过「小直播」单案例贯穿全书，从本地播放器逐步演进为支持万人连麦的直播平台。

---

## 这本书讲什么？

**不是**罗列 API 的说明书，**而是**带你经历真实的开发过程：

- 为什么需要这个功能？
- 几种方案怎么选？
- 遇到问题怎么调试？
- 如何优化性能和稳定性？

**单案例渐进式**：每一章在前一章代码基础上增加功能，始终可运行。


## 适合谁？

**适合：**
- 有 C++ 基础，但对音视频开发不熟悉
- 想系统学习直播技术，不只是零散搜博客
- 希望写出工业级代码，而不是玩具 demo

**不需要：**
- 音视频编解码基础（书中会讲）
- 直播开发经验（从零开始）

**前置知识：**
- C++ 基础语法（类、指针、STL）
- 命令行和 git 基本使用

---

## 技术栈

- **语言**：C++14
- **核心库**：FFmpeg 4.0+、SDL2
- **构建工具**：CMake 3.10+
- **支持平台**：macOS、Linux

---

## 快速开始（第1章）

```bash
git clone https://github.com/chapin666/live-system-book.git
cd live-system-book/chapter-01

# macOS 安装依赖
brew install ffmpeg sdl2 cmake

# 构建
mkdir build && cd build
cmake .. && make -j4

# 生成测试视频
ffmpeg -f lavfi -i testsrc=duration=10:size=640x480:rate=30 -pix_fmt yuv420p sample.mp4

# 运行
./live-player sample.mp4
```
