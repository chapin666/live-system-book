每一章在前一章**代码基础上增量开发**，不是孤立的 Demo。

**工业级代码标准**：
- RAII 智能指针，异常安全
- 详细错误码，不是简单返回 false
- 内置统计、日志、事件回调

**深入浅出讲解**：

| 概念 | 一句话解释 | 类比 |
|:---|:---|:---|
| Pipeline | 数据像水一样流动 | 工厂流水线 |
| PTS | 视频帧的"闹钟" | 告诉系统这帧该什么时候显示 |
| YUV | 把亮度和颜色分开存 | 黑白电视 + 彩色调色板 |
| SFU | 选择性转发 | 快递分拣中心 |
| MCU | 混音混画 | 视频编辑软件实时合成 |

## 🚀 快速开始

### 环境准备

**macOS:**
```bash
brew install ffmpeg sdl2 cmake pkg-config
```

**Ubuntu/Debian:**
```bash
sudo apt-get update
sudo apt-get install -y ffmpeg libavformat-dev libavcodec-dev \
    libavutil-dev libswscale-dev libsdl2-dev cmake pkg-config
```

### 构建运行（第 2 章 - 第一个播放器）

```bash
git clone https://github.com/chapin666/live-system-book.git
cd live-system-book/chapter-02

mkdir build && cd build
cmake .. && make -j4

# 生成测试视频
ffmpeg -f lavfi -i testsrc=duration=10:size=640x480:rate=30 \
       -pix_fmt yuv420p sample.mp4

# 运行
./player sample.mp4
```

**Part 1 完整运行路径**：

```bash
# 本地播放基础
cd chapter-02 && mkdir build && cd build && cmake .. && make && ./player video.mp4

# 工程化版本（Pipeline架构）
cd chapter-03 && mkdir build && cd build && cmake .. && make && ./player video.mp4

# 性能分析工具
cd chapter-04 && mkdir build && cd build && cmake .. && make
./timing_demo video.mp4        # 解码耗时统计
./lag_demo video.mp4 50        # 故意卡顿实验

# 多线程示例
cd chapter-05 && mkdir build && cd build && cmake .. && make
./thread_safe_queue_demo

# 异步播放器
cd chapter-06 && mkdir build && cd build && cmake .. && make && ./player video.mp4

# ... 以此类推
```
